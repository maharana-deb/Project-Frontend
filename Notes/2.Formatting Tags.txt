Formatting tags are used to format the text.
There are formatting tags like 
b -> used to make text bold
strong -> also used to make text bold but better SEO 
i -> used to make text italic,
em -> used to make text italic but better SEO
u -> used to make test get underlined
ins -> used to make text get underlined but it means
    inserted test to SEO
del -> used to make text deleted(strike through)
sub -> used to get subscript
sup -> used to get superscript
pre -> pre means pre-formatted text. Pre is used to
    bring pre formatted data.
br -> break the line
hr -> horizontal line
